# Pure-QR - Web Showcase 📱✨

This branch serves the [Pure-QR](https://github.com/janos-raul/pure-qr) GitHub Pages site.

**Pure-QR** is a minimalist, privacy-first Android application designed for generating and scanning QR codes and barcodes. It ensures all data stays on your device—no internet, no tracking, no compromise.

---

## 🌐 Live Page
Visit the live showcase here: [https://pure-qr.eu](https://pure-qr.eu/)

---

## 🚀 Key Features of the App

### 🔍 Smart Scanner & Inspector
- **Auto-Detection:** Seamlessly switches between QR and 1D Barcode modes.
- **Auto-Zoom:** Intelligent focus using Google ML Kit.
- **Payload Inspector:** Audit raw code content (UTM, ref, etc.) instantly.

### ✍️ Advanced Generator
Generate high-quality QR codes for:
- 📝 **Plain Text** & 🔗 **URLs**
- 📶 **WiFi Networks**
- 📇 **Contacts** (vCard)
- 💰 **Crypto Addresses**
- 📊 **1D Barcodes**

### 🎨 Deep Customization
- **Offline Branding:** Custom Foreground, Background, and Frame colors.
- **Dynamic Frames:** Multiple styles (Rounded, Thick, Modern, Dotted).
- **Smart Icons:** Automatic center logos for WiFi, V-Cards, and Crypto.

## 🔒 Privacy Commitment

- **100% Offline:** No internet permissions requested.
- **Zero Tracking:** No analytics, no ads, no background services.
